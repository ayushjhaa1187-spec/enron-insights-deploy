# Enron Insights Deploy

> Production deployment infrastructure for the Enron Insights forensic data analysis platform.

![build](https://img.shields.io/badge/build-passing-brightgreen) ![license](https://img.shields.io/badge/license-MIT-blue) ![stack](https://img.shields.io/badge/stack-TypeScript%20%7C%20Vite%20%7C%20Supabase-blueviolet)

## 📖 The Story
> *"The forensic engine was powerful, but it sat in a local lab, inaccessible. The investigation needed scale. He ran the Deploy script. One cluster. One cloud. Enron Insights went global in minutes. The auditors finally had the tools they needed."*

## ✨ Features
| Feature | Description |
| :--- | :--- |
| **Production-Ready Build** | Optimized Vite build with SSR and code-splitting |
| **Supabase Integration** | Real-time database and auth for forensic data queries |
| **CI/CD Pipeline** | Automated deployment via Vercel with preview environments |
| **Secure Data Handling** | Encrypted partitioning for sensitive forensic datasets |

## 🛠 Tech Stack
- **Frontend**: Vite, React, TypeScript
- **UI**: Tailwind CSS, Shadcn UI
- **Backend**: Supabase
- **Deployment**: Vercel

## 🚀 Quick Start
```bash
git clone https://github.com/ayushjhaa1187-spec/enron-insights-deploy.git
cd enron-insights-deploy
npm install
npm run dev
```
