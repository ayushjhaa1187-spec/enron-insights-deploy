# enron-insights-deploy
Automated deployment infrastructure for the Enron Insights forensic platform.

"The forensic engine was powerful, but it sat in a local lab, inaccessible. The investigation needed scale. He ran the Deploy script. One cluster. One cloud. Enron Insights went global in minutes. The auditors finally had the tools they needed."

## Tech Stack
Terraform, AWS, Docker, GitHub Actions

## What This Does
Handles the end-to-end orchestration of the Enron Insights platform, ensuring high availability and secure data partitioning for sensitive forensic queries.
